DOS-866

Описание параметров для запуска программы Palitra

$ запуск с параметром N
------------------------------------------------------------------------------+
Цвета по умолчанию будут соответствовать переданным в параметрах. 
Возможны варианты запуска:

  /sys/media/palitra N 00FF00FF 00FF0000
  /sys/media/palitra N 00F724A3 00736698
..............................................................................+


$ запуск с параметром H 
------------------------------------------------------------------------------+
Программа запускается в hidden mode без отрисовки окна приложения.
После чего, цвета переданные в параметре будут использованы для заливки фона 
рабочего стола градиентом из указанных цветов.

  /sys/media/palitra H 00FF00FF 00FF0000
  /sys/media/palitra H 00F724A3 00736698
..............................................................................+
